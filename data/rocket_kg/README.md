# ROCKET-KG Data

## Overview

ROCKET-KG is the multi-source biomedical knowledge graph that underpins
the ROCKET framework.  It aggregates three information sources:

| Source | Method | Coverage |
|--------|--------|---------|
| **LLM-generated triples** | GPT-3.5 / GPT-4 prompted with each medical code | 285 conditions + 231 procedures + 269 drugs |
| **UMLS** | 2-hop subgraph extraction around each medical concept | Enriches entity relations with biomedical ontology |
| **Rocket ontology** | Structured medical vocabulary (ICD → CCS hierarchy) | Adds code hierarchy edges |

## Triple File Format

Each `.txt` file contains ~100 tab-separated triples:

```
head_entity\trelation\ttail_entity
```

Example (`condition/CCSCM/1.txt` — septicemia):
```
septicemia    is caused by    bacterial infection
septicemia    is treated with    antibiotics
septicemia    is a risk factor for    organ failure
blood culture    is a diagnostic test for    septicemia
```

## Building the KG from Scratch

```python
from src.kg_construction import KGBuilder, EmbeddingBuilder, ClusteringPipeline

# Step 1: Generate triples
builder = KGBuilder(openai_api_key="<ADD_YOUR_OPENAI_KEY_HERE>")  # ← Add your OpenAI API key here
builder.build_from_csv("resources/CCSCM.csv", "code", "name", "condition", "CCSCM")
builder.build_from_csv("resources/CCSPROC.csv", "code", "name", "procedure", "CCSPROC")
builder.build_from_csv("resources/ATC.csv", "code", "name", "drug", "ATC3")

# Step 2: Build embeddings
eb = EmbeddingBuilder(openai_api_key="<ADD_YOUR_OPENAI_KEY_HERE>")  # ← Add your OpenAI API key here
eb.build("data/rocket_kg/graphs/condition/CCSCM")
eb.build("data/rocket_kg/graphs/procedure/CCSPROC")
eb.build("data/rocket_kg/graphs/drug/ATC3")

# Merge for each task group
eb.merge(
    dirs=["graphs/condition/CCSCM", "graphs/procedure/CCSPROC"],
    out_dir="graphs/merged/CCSCM_CCSPROC",
)
eb.merge(
    dirs=["graphs/condition/CCSCM", "graphs/procedure/CCSPROC", "graphs/drug/ATC3"],
    out_dir="graphs/merged/CCSCM_CCSPROC_ATC3",
)

# Step 3: Cluster
pipe = ClusteringPipeline(threshold=0.15)
pipe.run(
    graphs_dir="graphs/merged/CCSCM_CCSPROC",
    triple_dirs={"ccscm": "graphs/condition/CCSCM", "ccsproc": "graphs/procedure/CCSPROC"},
    out_dir="clustering/ccscm_ccsproc",
)
```
